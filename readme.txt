Accesses the CafePress API via shortcodes to display millions of products based on search terms and product types. PHP5 is required.

== Description ==

Whether you want to display your own products as a shopkeeper using your unique tags or you want to sell other's 
products for a commission, this plugin is for you.

This release fixes directory issues in the original release 1.0.

Features:

* No REV SHARE. All commissions are yours. If you make money, send some our way.
* Option of displaying products by row or fluid thumbnails
* Option of linking to the marketplace or the shop version of the product
* Option to show the design link
* Option to show the shop name and link
* Option to use the Commission Junction-styled links or CafePress-styled links
* Fields for your Commission Junction PID, SID and XID information
* Display is fully customizeable with CSS
* Includes link to the Cafewish easy [product type selector](http://www.cafewish.com/cpgrid_products.asp "product type selector")
* Reviewed and approved by CafePress
 

== Installation ==

1. Unzip and upload all files and subdirectories to the `/wp-content/plugins/wishads-for-cafepress-search` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set up the plugin with your Commission Junction and CafePress information in the `Settings` area
4. Use the shortcode or the input form to create your product display.
5. A help file is included with more information about affiliating CafePress

== Frequently Asked Questions ==

= Do I have to have a CafePress store to sell products? =

No. You can join the CafePress affiliate program without being a shopkeeper.

= Do I have to be involved in the Commission Junction program for CafePress to show my own shop products? =

If you are a CafePress shopkeeper, you can show your own products by using a unique tag on your designs. 
Leave off the affiliate information and it will link directly to your shop without having to be an affiliate.

= Is this one of those 4th-click kind of plugins where we share in revenues? =

No, all revenues and clicks belong to you.
